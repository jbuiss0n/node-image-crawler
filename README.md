node-image-crawler
===============

> A node.js iweb image crawler.

Requirements
------------

* [node.js](http://nodejs.org/)

Install
------------

  npm install

Test (there is no test at the moment, and it's a shame...)
------------

  npm test

Usage
------------

  node server.js
